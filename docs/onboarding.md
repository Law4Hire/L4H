## Welcome

This guide helps new contributors get up and running with Law4Hire and Cannlaw.

## Prerequisites

- .NET 10 SDK (pre-release)
- Node.js + Yarn
- Docker Desktop
- Git + GitHub access

## Setup Steps

1. Clone repo
2. Run `docker-compose up`
3. Install frontend deps: `yarn install`
4. Run tests: `dotnet test`, `yarn test`
